# 共享卡组

您可以访问Anki卡组库以及Ankiweb获取网络公开卡组 如果您想与有限群体（例如学习小组或班级）而不是普通公众共享套牌，您可以通过在AnkiWeb之外共享它们来实现此目的。

要私下共享一个套牌，请转到“文件”菜单并选择“导出”。选择单个牌组（不是“All Decks”），然后关闭“包括排程信息”。这将生成一个.apkg文件，您可以与他人共享。

您可以通过向人们发送电子邮件，将其放在网站或共享文件夹中，或使用Dropbox或Google Drive等免费文件共享服务并向人们发送链接来共享.apkg文件。

计算机版和移动客户端都可以通过单击或点击它来轻松从apkg文件导入。但是，AnkiWeb无法导入apkg文件，因此您的卡组的收件人需要在其移动设备上安装计算机版本或Anki。

当用户导入.apkg文件时，将忽略其集合中已存在的卡，并添加任何新卡。只要他们使用相同的笔记类型，修改后的卡片也会更新。为防止数据丢失，新apkg文件中已删除的卡不会在用户的集合中删除，因此如果您因任何原因需要从用户的卡片中删除卡，则需要与他们联系。
