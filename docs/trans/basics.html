# 基本术语

# 卡片

一问一答配对成卡片。卡片的一面是问题，另外一面是答案。在Anki里的卡片不像是实体卡那样，答案并不是显而易见的。例如，你学习基础化学，你可能会看到这样的问题：

问题：氧气的化学符号？

经过思考后，你选择答案为正确，点击显示按钮，则会显示你的回答：

问题：氧气的化学符号？
答案：O

在确认你是正确的同时，你也知道了你的记忆情况，Anki会根据你对这个问题的记忆情况选择一个时间让这个问题再次出现。

# 牌组

一副牌组包含一组卡片。可以替换卡片组成不同的牌组去学习而无需每一次都学习所有的内容。每个牌组可以有不同的设置，如每天展示多少卡片，亦或是多长时间后卡片再次出现。

# 笔记＆区域

制作卡牌时，通常需要制作多张与某些信息相关的卡片，即需要为某个知识点制作多个卡片。例如，如果您正在学习法语，并且了解到单词“bonjour”的意思是“你好”，那么您可能希望创建一张显示“bonjour”并使其令你记住“你好”的卡片，另一张显示“你好”并使其令你记住“bonjour”的卡片。一张卡片是测试你识别外语单词的能力，另一张卡片是测试你产出能力。

当使用纸质抽认卡时，在这种情况下，你唯一的选择是将知识点写两遍，每张卡片一次。一些电脑抽认卡程序通过提供翻转正面和背面的功能，使学习变得更容易。这比纸面情况有所改善，但有两个主要缺点:

  • 因为这类程序不会单独跟踪你的识别和生产表现，所以卡片往往不会在最佳时间显示给你，这意味着你忘记的东西比你想要的多，或者你学习的东西比需要的多。

  • 只有当你想让两边的内容完全相同时，翻转卡片才有效。这意味着不可能在每张卡片的背面显示额外的信息。

通过把卡片的内容分离成独立的几条信息，Anki 解决了以上问题。你告诉 Anki 每张卡片所需要的信息，Anki 为你自动创建这些卡片，并在后续你编辑之后更新卡片。

假如我们想学习法语单词，而且我们希望在每张卡片的背面附上页码。我们期望的卡片是这样的：

问：Bonjour
答：你好
页码：12

和：

问：你好
答: Bonjour
页码：12

这个例子里，存在三条三关的信息：一个法语单词，一个中文释义，一个页码。如果将它们放在一起，则是这样的：

法语：Bonjour
中文：你好
页码：12

Anki 中这些相关的信息称为笔记，每条信息称为字段。所以我们可以说这里的笔记有三个字段：法语，中文和页码。

要增加或编辑字段，点击新增和编辑笔记时的“字段”按钮。更多字段相关的信息，参见定制区域小节。

# 卡片类型

为了让 Anki 根据我们的笔记创建卡片，我们需要给它一个蓝图，说明应该在每张卡片的正面或背面显示哪些字段。这个蓝图称为卡片类型。每种类型的便条可以有一种或多种卡片类型;当您添加注释时，Anki将为每种卡片类型创建一张卡片。

每种卡片类型都有两个模板，一个用于回答问题，另一个用于回答问题。在上面的法语示例中，我们希望识别卡是这样的:

问：Boujour
答：你好
页码：12

问了达到此目标，我们可以把问题模版和答案模版设置成这样的：

问: {{法语}}
答: {{英语}}<br>
页码: #{{页码}}

通过将字段名放在两对大括号里，我们告诉 Anki 将括号里的内容替换为实际的字段内容。每一张卡片上没有被括号包围的东西。（例如，我们不需要类型”页#”进入页面字段添加字段–是自动添加到每个卡。）<br>是一个特殊的代码，告诉Anki移动到下一行；更多的细节在卡片与模板部分。

制作卡片模板方法类似：

问: {{法语}}
答: {{英语}}<br>
页码: #{{页码}}

一旦创建了卡片类型，每次添加新笔记时，将创建一个基于该卡片类型的卡片。卡片类型让卡片格式的一致性更容易保持，并且大大减少了新增卡片所需的精力。这同时意味着相关联卡片的出现不会过于集中，也让你更正笔误等错误后所有的卡片都会得到更新。

要编辑卡片类型，点击新增和编辑笔记时的“卡片”按钮。更多与卡片类型相关的信息请参见卡片与模板小节。

# 笔记类型

Anki 允许你为不同的材料创建不同类型的笔记。每种类型的便条都有自己的一组字段和卡片类型。为你正在学习的每个主题创建一个单独的笔记类型是一个好主意。在上面的法语示例中，我们可以为此创建一个名为“French”的注释类型。如果我们想学习首都城市，我们也可以为它创建一个单独的注释类型，带有“Country”和“Capital City”等字段。

Anki 检查重复项时只会在笔记类型相同的卡片中比较。所以如果你新建了一个首都叫“Orange”的各国首都类型的卡片，下次你学法语里的“Orange”时并不会看到卡片重复的提示。

当您创建一个新的集合时，Anki 会自动向其中添加一些标准的注释类型。提供这些笔记类型是为了让新用户更容易使用Anki，但从长远来看，建议您为正在学习的内容定义自己的笔记类型。标准卡片类型如下所示：

Basic

每张卡片有 Front（正面） 和 Back（背面）两个字段。Front 字段输入的文本显示在卡片正面，Back 字段的文本显示在背面。

Basic（翻转卡片）

和 Basic 类似，但是会创建两张卡片，一张是 Front 到 Back, 另一张是 Back 到 Front。

Basic（可选翻转卡片）

此类型包括一张 Front 到 Back 的卡片，和另一个可选的 Back 到 Front 的卡片。为了实现此功能，它拥有第三个字段“Add Reverse”。在此字段输入任意内容，就会另外创建一张翻转卡片。更多内容请参见卡片与模板小节。

Cloze

此笔记类型让选择文本并挖空更加简单（例如，“人类在[...]年登月”->“人类在1969年登月”。更多内容，参加填空小节。

新增或修改卡片类型，请使用“工具”->“管理笔记类型”功能。

注意

笔记与笔记类型可以在整个集合范围内公用，而不是局限于特定的牌组。这意味着特定的牌组内部可以使用多种类型的笔记，某个特定的笔记所生成的多个卡片也可以放到不同牌组中。使用“添加”窗口创建卡片时，你可以选择要使用的卡片类型与牌组，这些选项之间完全互不干扰。 你也能修改已有卡片的笔记类型。

# 集合

集合是Anki 所存储的材料的总和，包括卡片、笔记、牌组、笔记类型、牌组选项等等。
