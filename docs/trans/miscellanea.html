# 菜单快捷方式

在Windows / Linux你可以按住ALT键并按一个突出字母激活特定的菜单。

OS X不支持此功能，但它允许你指定快捷方式到特定的菜单项。更多信息请见http://lifehacker.com/343328/create-a-keyboard-shortcut-for-any-menu-action-in-any-program

# 调试控制台(debug)

有时可能会要求您使用debug控制台更改设置或检查某些内容。除非要求在“调试控制台”中输入文本，否则您可能不需要这样做。高级用户可能喜欢在下面的附加组件页面中阅读更多关于它的信息，链接如下。

当被要求在“调试控制台”中输入文本时，请启动Anki，并在主窗口中按下

Ctrl + Shift +;

(同时控制键、shift键和分号键)

在Mac电脑上，按下

命令+ Shift +;

(命令键、shift键和分号键同时使用)

在一些非英语键盘上，你可能需要按“:”或“+”而不是“;”。

在弹出的窗口中，请粘贴您被要求粘贴在顶部部分的文本。当你这样做了，请按Ctrl+Return(命令+ Mac返回)，一些文本应该出现在底部部分。如果您被要求粘贴结果输出，请从底部区域复制它，并将其粘贴至社区寻求帮助。

如果您按Ctrl+Shift+Return而不是Ctrl+Return, Anki 将尝试打印结果，而不是按您要求的那样做。如果您遇到意外错误，请确保您没有按住Shift键。

# 插件

Anki 的功能可以通过附加组件进行扩展。附加组件可以提供诸如对特定语言的额外支持、对调度的额外控制等特性。

要浏览可用外接程序列表，请选择工具>外接程序菜单项，然后单击“获取插件”。

许多附加组件的作者都在附加组件中包含他们的电子邮件地址，所以如果您需要与作者联系，编辑附加组件并查看文件的顶部可能会有所帮助。

如果您下载的附加组件不能正常工作，或者在编辑附加组件时意外出错，您可以使用菜单中的“删除”选项删除它。

若要学习如何编写自己的附加组件，请参阅插件书写指南
