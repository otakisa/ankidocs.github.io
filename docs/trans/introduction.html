# 介绍

Anki是一个使记忆变得容易的程序，或者说是学习软件。因为它是具有很强的灵活性，比传统的学习方法更加有效率，可以大大减少你的学习时间，或者大大提高你的学习数量.

ANKI可以帮助任何需要在日常生活中记忆东西的人们。因为它与内容无关，支持图像、音频、视频和科学标记（通过LaTeX），因此具有无限可能性。 例如:

  • 学习一门语言
  • 备考医学和法律考试
  • 记忆人的名字和面孔
  • 温习地理学
  • 学习掌握长诗
  • 即便是学习吉他和弦

Anki背后有两个简单的概念：主动回忆学习和间隔重复。尽管已经在科学文献中写了很多年，但大多数学习者并不了解它们。了解它们的工作方式将使您成为更有效的学习者。

# 主动回忆测试

主动回忆测试的意义是：被问到一个问题的同时并试图记住答案。这与被动学习形成鲜明对比，在被动学习中，如果我们知道答案，我们会阅读，观看或聆听某些内容，而不会停下来考虑。研究表明，积极主动回忆学习在构建强大记忆方面比被动学习更有效。有两个原因：

  • 回忆某事的行为增强了记忆力，增加了我们再次记住它的机会。
  • 当我们无法回答问题时，它告诉我们需要返回材料进行审核或重新学习。

你可能在学校里遇到过主动回忆测试，但你甚至没有意识到。当一些好老师在你读完一篇文章后给你一系列的问题要你回答，或者让你每周做一次进度检查测试，他们这么做并不是为了看你是否理解了材料。通过测试你，他们增加了你将来能记住材料的机会。

将主动回忆测试融入你自己的研究的一个好方法是使用抽认卡。用传统的纸质抽认卡，你在卡片的一面写一个问题，另一面写答案。在你想好答案之前不要翻开卡片，你可以比被动观察更有效地学习。

# 使用它不然就会忘记

我们的大脑是高效的机器，它们会迅速地丢弃那些看起来无用的信息。很可能你不记得两周前周一吃了什么，因为这些信息通常没什么用。然而，如果你那天去了一家很棒的餐厅，花了最后两周的时间向人们讲述它有多棒，你很可能仍能生动地记住细节。

大脑的“使用它或失去它”策略适用于我们所学的一切。如果你花一个下午的时间来记忆一些科学术语，然后两周内不去想那些材料，你可能已经忘记了大部分。事实上，研究表明我们会在48小时内忘记75%的学习内容。当你需要学习很多信息的时候，这听起来很令人沮丧。

然而，解决方案很简单:回顾。通过复习新学过知识，我们可以大大减少遗忘。

唯一的问题是传统的复习不是很实用。如果你用的是纸质抽认卡，如果你只有30张要复习的话，很容易就能快速浏览完所有的卡片，但是随着抽认卡的数量增加到300或3000张，它很快就变得难以处理了。

# 间隔重复

1885年，一位德国心理学家报告了间隔效应。他发现，如果我们把复习时间分散开来，而不是在一段时间内学习多次，我们往往能更有效地记住东西。自20世纪30年代以来，有很多人提出利用间隔效应来提高学习效果，这就是所谓的间隔重复。

一个例子是1972年，一位名叫塞巴斯蒂安·莱特纳(Sebastian Leitner)的德国科学家推广了一种用抽认卡进行间隔重复的方法。通过将纸质卡片分成一系列的盒子，并在每次成功或失败的学习中将卡片移动到不同的盒子中，可以粗略地估计出一张卡片的熟悉程度以及何时应该复习。这是一个很大的改进，这比单一的一盒卡更加有效，它已被广泛采用的电脑抽认卡软件。然而，这是一个相当粗略的方法，因为它不能给你一个确切的日期，你应该在什么时候再复习一遍，而且它也不能很好地处理不同难度的材料。

过去30年里最大的进展来自SuperMemo的作者，这是一个商业抽认卡程序，实现了间隔重复。SuperMemo首创了一个系统的概念，该系统可以跟踪查看材料的理想时间，并根据用户的表现优化系统本身。

在SuperMemo的间隔重复系统中，每次你回答一个问题，你都要告诉程序你的记忆力有多好——你是否完全忘记了；犯了一个小错误；记起来很麻烦；很容易记起来；等等。程序使用这个反馈来决定再次显示问题的最佳时间。因为每次你成功地回忆起一个问题，记忆就会变得更强，所以两次复习之间的间隔时间就会变得越来越长——所以你可能第一次看到一个问题，然后3天，15天，45天，等等。

这是学习上的一场革命，因为它意味着学习和记忆材料只需要最少的努力便可实现。SuperMemo的口号总结了这一点:通过间隔重复，你可以忘记‘忘记’。

# 为什么选择ANKI?

虽然不可否认SuperMemo对这个领域产生了巨大的影响，但它也并非没有问题。该程序经常被批评错误百出，难以调试，并且它只能在Windows电脑上运行。它是专有软件，这意味着终端用户不能扩展它或访问原始数据。虽然非常旧的版本是免费提供的，但它们对现代使用非常有限。

Anki解决了这些问题。在许多平台上都有免费的Anki客户端，所以预算紧张的学生和老师也不会被排除在外。它是开源的，终端用户贡献的插件库已经非常丰富；它是多平台的，运行在Windows、Mac OSX、Linux/FreeBSD和一些移动设备上。它比SuperMemo更容易使用。

Anki的间隔重复系统是基于旧版本的超级备忘录算法SM-2。
