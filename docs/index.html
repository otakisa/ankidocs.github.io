# 简单介绍

# Anki到底是什么

Anki是一个工具，本质上就是一个电子抽认卡的程序，可以对每个卡牌的内容进行自定义并自我检测学习后，程序会通过算法，在必要的时间让你进行复习，从而达到滚动、高效的复习。且由于其开源的特点，人人可用，人人可以二次开发，这就使得Anki环境更加优秀。

# Anki可以做什么

Anki是一个按照“艾宾浩斯曲线”理论编写的的记忆复习软件，提供问答模式来使得学习者更好的学习与复习，你可以用这款软件进行多国语言的学习、学习并复习知识点以及复习相关材料使用。

# Anki有哪些特点

  • 1、结合了遗忘规律，能够智能化的为你安排复习回忆时间；
  • 2、随时可以复习，充分利用一些琐碎的时间；
  • 3、本身有很多开放的词库,包括本站提供的Anki卡组库以及官方提供的Ankiweb,因此它的多样性使得你可以学习任何语言或其他知识
  • 4、Anki是跨平台的自由软件系统，Anki提供了Windows,Linux,Mac,Android,iPhone, 及在线版，本文下方也给出了下载地址。Anki的多版本使得它几乎支持所有主流操作系统版本。Anki还具有免费的网络同步功能，只要注册一个账号即可在不同系统间同步你的卡组内容，避免你进行重复收录卡组的无谓工作，非常节省时间。这里给出在线服务注册地址Ankiweb
